A little Jump & Run game where you compete against other players, additionally there are modifiers to make your run easier and the others harder.

## About this game
In this game you compete against others players in a multiplayer Jump & Run game. There is one large map with different blocks, like jump pads, moving blocks and more... Additionally there a random modifiers you can collect, some make you as a player stronger & give you advantages, others nerf the others. There are modifiers like slowing, or slippery ice roads, when slippery ice roads is active, players can no longer decelerate normally.
> [!NOTE]
> All the texts are in german

## Why
I made this game as a present for my father's birthday in 2024, this is also why the game is called **Platform Dad**. I made this game multiplayer to be able to play this game together with my father. At the end, it all went wrong, because I (of course) knew exactly how the game works and were more skilled than my father who never played it before.

## The experience
Making this project was really fun, as it just took several hours, it was a quick refresh to get my head free from my other big project at the time (Kitchen Quest). As I already noticed in other projects, after disabling VSM & Nanite, the game started more quickly and the runtime performance was better than before.

## Play the game yourself
Currently, I don't have any prebuild builds. If you want to play the game, you have to build UE 5.7 from source and build the dedicated server & client. Prebuild Windows & Linux files will come soon.
